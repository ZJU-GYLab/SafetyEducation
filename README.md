# GYLab 安全教育

这是一个基于网页的实验室安全教育平台，旨在为实验室成员提供全面的安全教育内容。

## 功能特点

- 响应式设计，支持桌面和移动设备
- 智能目录导航系统
- 清晰的内容结构
- 搜索功能

## 使用方法

1. 克隆仓库到本地：
```bash
git clone [your-repository-url]
```

2. 在浏览器中打开 `index.html` 文件

## 部署

本项目可以直接部署到 GitHub Pages：

1. 创建一个新的 GitHub 仓库
2. 将代码推送到仓库
3. 在仓库设置中启用 GitHub Pages

## 技术栈

- HTML5
- CSS3
- JavaScript
- DocumentOutline.js（自定义导航组件）
